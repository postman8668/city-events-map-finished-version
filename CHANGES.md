# Изменения в системе событий

## 1. Добавлена система модерации событий

### Что изменилось:

1. **Создание событий для всех пользователей**
   - Теперь любой авторизованный пользователь может создавать события
   - Доступ к форме создания: `/events/create` или через меню "Мои события" → "Создать событие"

2. **Статусы событий**
   - `pending` - событие на модерации (создано обычным пользователем)
   - `approved` - событие одобрено и опубликовано
   - `rejected` - событие отклонено администратором

3. **Права пользователей**
   - **Обычные пользователи:**
     - Могут создавать события (отправляются на модерацию)
     - Могут редактировать свои события
     - Видят свои события в разделе "Мои события"
   
   - **Администратор:**
     - События создаются сразу одобренными
     - Может модерировать события (одобрять/отклонять)
     - Может редактировать и удалять любые события
     - Видит все события с фильтрацией по статусу

4. **Новые страницы**
   - `/my-events` - страница "Мои события" для просмотра созданных событий
   - `/events/create` - форма создания события (доступна всем авторизованным)
   - `/events/<id>/edit` - редактирование события (автор или админ)

5. **Админ-панель**
   - Добавлен счетчик событий на модерации
   - Фильтрация событий по статусу (все/на модерации/одобренные/отклоненные)
   - Кнопки "Одобрить" и "Отклонить" для событий на модерации

6. **Изменения в базе данных**
   - Добавлено поле `status` (статус события)
   - Добавлено поле `creator_id` (автор события)
   - Все существующие события получили статус `approved`

## 2. Интерактивная карта для выбора координат

### Что добавлено:

1. **Интерактивная карта в формах**
   - Карта встроена в формы создания и редактирования событий
   - Использует Leaflet.js и OpenStreetMap
   - Размер карты: 400px в высоту

2. **Выбор места кликом**
   - Кликните на карте - метка установится в этой точке
   - Координаты заполнятся автоматически
   - Точность: 6 знаков после запятой

3. **Перетаскивание метки**
   - Метку можно перетаскивать мышью
   - Координаты обновляются в реальном времени
   - Всплывающая подсказка показывает координаты

4. **Автоматическое заполнение**
   - Поля "Широта" и "Долгота" заполняются автоматически
   - Поля доступны только для чтения (readonly)
   - Исключены ошибки ручного ввода

5. **Визуальная обратная связь**
   - Всплывающая подсказка на метке
   - Показывает "Место события" и координаты
   - Автоматически открывается при установке метки

### Где работает:

- ✅ `/events/create` - создание события
- ✅ `/events/<id>/edit` - редактирование события

### Преимущества:

- 🎯 Точный визуальный выбор места
- 🚀 Не нужно искать координаты вручную
- 👁️ Наглядное расположение на карте
- ✏️ Гибкая настройка перетаскиванием
- 🔒 Защита от ошибок ввода

## Как использовать:

### Для обычных пользователей:

#### Создание события с картой:
1. Войдите в систему
2. Перейдите в "Мои события" → "Создать событие"
3. Заполните название, описание, дату и время
4. Прокрутите до карты
5. **Кликните на карте** в месте проведения события
6. Метка установится, координаты заполнятся автоматически
7. При необходимости перетащите метку для точной настройки
8. Заполните остальные поля и отправьте
9. Событие появится со статусом "На модерации"

### Для администратора:

#### Модерация событий:
1. Войдите как администратор
2. Перейдите в "Админ" → "События"
3. Используйте фильтр "На модерации"
4. Нажмите "Одобрить" или "Отклонить"

#### Создание события с картой:
1. Перейдите в "Админ" → "События" → "Создать событие"
2. Заполните форму
3. **Кликните на карте** для выбора места
4. Событие будет опубликовано сразу (без модерации)

## Технические детали:

### Модерация:
- Модель `Event` расширена полями `status` и `creator_id`
- Добавлена связь `Event.creator` → `User`
- API `/api/events` возвращает только одобренные события
- Главная страница показывает только одобренные события
- Логирование всех действий с событиями

### Интерактивная карта:
- Библиотека: Leaflet.js 1.9.4
- Карты: OpenStreetMap
- Начальная позиция: Полоцк (55.485, 28.768)
- Зум: 12 (создание), 15 (редактирование)
- Метка: перетаскиваемая (draggable)
- Координаты: readonly поля, автозаполнение
