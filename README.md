# 🗺️ Карта событий Полоцка и Новополоцка

Веб-приложение для агрегации и навигации по городским мероприятиям с использованием интерактивных картографических сервисов.

## Возможности

- 🗺️ Интерактивная карта с маркерами событий (Leaflet.js)
- 🔍 Поиск и фильтрация по категориям, интересам, цене
- 👥 Присоединение к событиям с ограничением участников
- ⭐ Система отзывов и рейтингов
- 📸 Загрузка изображений для событий
- 🔔 Уведомления для пользователей
- 👨‍💼 Роли: администратор, модератор, пользователь
- 📊 Админ-панель с модерацией и статистикой
- 📱 Адаптивный дизайн для мобильных устройств

## Технологии

Python 3.10, Flask, SQLAlchemy, SQLite, Bootstrap 5, Leaflet.js, Toastify.js

## Установка

```bash
git clone https://github.com/postman8668/city-events-map-finished-version.git
cd city-events-map-finished-version
pip install -r requirements.txt
```

Создайте файл `.env`:
```
SECRET_KEY=your-secret-key
MAIL_USERNAME=your-email@gmail.com
MAIL_PASSWORD=your-app-password
```

Запуск:
```bash
python app.py
```

Приложение будет доступно по адресу: http://127.0.0.1:5000

## Роли пользователей

- **Admin** - полный доступ к системе, управление пользователями и событиями
- **Moderator** - модерация событий, просмотр логов
- **User** - создание событий, присоединение, отзывы
