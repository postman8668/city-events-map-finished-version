# 🗺️ Карта событий Полоцка и Новополоцка

Веб-приложение для поиска и публикации событий в городах Полоцк и Новополоцк с интерактивной картой.

## Возможности

- Поиск событий на интерактивной карте
- Фильтрация по категориям и интересам
- Присоединение к событиям и отзывы
- Создание собственных событий
- Админ-панель с модерацией

## Технологии

Python 3.10, Flask, SQLite, Bootstrap 5, Leaflet.js

## Установка

```bash
git clone https://github.com/postman8668/city-events-map-finished-version.git
cd city-events-map-finished-version
pip install -r requirements.txt
python app.py
```
